Personal Portfolio Website

A modern, fully-responsive portfolio site built with plain HTML, CSS, and vanilla JavaScript. It showcases this portfolio website with skills, experience, and projects while remaining easy to extend and maintain.

Key Highlights

  • Static & Lightweight – No frameworks; deploy instantly to GitHub Pages, Netlify, or any static host.
  • Modular Layout – header.html and footer.html are injected on every page via include.js, ensuring site-wide consistency.
  • Responsive Navigation – Adaptive header with burger menu for mobile; active-link highlighting handled in JS.
  • Clean Sectioning – About, Skills, Experience on home; Project cards on projects page; Code samples on this page.
  • Single-file Styling – styles.css uses CSS variables, flex/grid, and media queries for a polished look.
  • Deployment-ready – Optional CNAME for custom domains; no back-end or build step required.

DataBreaker (Unity Action-Game Framework)

A modular, third-person action game framework built in Unity (C#) with a ScriptableObject-driven architecture. Designed for fast-paced traversal and combat, it cleanly separates gameplay concerns into interchangeable systems and data containers—making it easy to extend, test, and iterate.

Key Highlights

  • Modular Systems – Checkpoints & respawn flow, combat combos & hit-detection, “Karnage” special modes, and global coordination via a central GameSystemsController.
  • Player Actions – Each move (dash, jump, grind, grapple) and attack (light/heavy) is its own ScriptableObject, allowing designers to tweak behaviors without touching code.
  • Camera Framework – Multi-mode setup (free, targeted, grind) with a reset system that smoothly repositions behind the player. Abstract base classes make adding new modes straightforward.
  • Data-Driven Design – DataContainer_Player and DataContainer_Camera store runtime state, settings, and references—decoupling logic from configuration.
  • Scalable Architecture – Clean folder structure (GameSystems/, MainCharacter/Actions/, Controllers/, DataContainers/) and thorough inline documentation support rapid onboarding.

Tech Stack & Tools

  • Unity 2021 LTS · C# 9
  • ScriptableObjects for behaviour modularity
  • Built-in Unity UI & Input System
  • Git for source control
